## 🎉 使用方式

### 启动文档服务
```bash
# 开发模式
pnpm run docs:dev

# 构建文档
pnpm run docs:build

# 预览构建结果
pnpm run docs:preview
```

### 访问地址
- **本地开发**: http://localhost:5173/
- **构建输出**: `docs/.vitepress/dist/`

### 文档维护
- **添加新页面**: 在相应目录创建 `.md` 文件
- **更新导航**: 修改 `docs/.vitepress/config.ts`
- **自定义样式**: 修改 `docs/.vitepress/theme/`
